Lobby system for online games

## Installation

### Using npm

```bash
npm i @jgtools/lobbyserver
```

```javascript
import LobbyServer from "@jgtools/lobbyserver";
// ...
```

### Using cdn

```html
<script type="module">
    import LobbyServer from "https://cdn.jsdelivr.net/npm/@jgtools/lobbyserver@1.0.0/dist/index.min.js";
    // ...
</script>
```

## Usage
